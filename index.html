lookup by jackowayed

A ruby program that looks up words on thefreedictionary.com

I made this because I had to lookup a list of a words and didn't want to do it manually. The file takes a list of words, each on its own line, in a file called in_words.txt in the same dir as lookup.rb. Then you just run: ./lookup.rb and it spits out the definitions in a file called definitions.rb, in the format word: definition

Dependencies

Ruby

Install

Just download lookup.rb from the repo. Make sure you also have Ruby installed (http://ruby-lang.org/)
